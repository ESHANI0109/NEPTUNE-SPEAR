# AI Care – Smart Health Monitoring System for Elderly

## Problem Statement
Many elderly people live alone and caretakers cannot constantly monitor their health, medicine schedule, meals, or safety. Missing medicines or abnormal behaviour can lead to serious health risks.

## Proposed Solution
We are building an AI-powered web platform that helps caretakers monitor elderly people remotely.

The system will track:
- Daily meal updates
- Medicine schedule
- Location tracking
- Daily health status

If the user fails to update details or if abnormal patterns are detected, the system will automatically send alerts to the caretaker.

## Features
- Caretaker dashboard
- Medicine reminder system
- Meal tracking
- Location monitoring
- AI-based anomaly detection
- Daily health report sent to caretaker

## Tech Stack
Frontend: HTML, CSS, JavaScript  
Backend: Python / Node.js  
Database: Firebase / MongoDB  
AI Model: Machine Learning anomaly detection

## Future Scope
- Integration with smartwatches
- Voice assistant support for elderly users
- Emergency medical alert system
- Live CCTV Footage in caretaker dashboard
- Medicines and provisions can be ordered online through the dashboard
